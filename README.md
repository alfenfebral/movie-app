## React Movie App

Simple application to search for movies (themoviedb).

### Instructions

replace <API_KEY> to your api key in config/index.js

### Dependencies

- redux-saga
- react-redux
- redux
- eslint (airbnb)
- prettier
- husky
- pretty-quick
- react-debounce-input
- prop-types
